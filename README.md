# ActiveDirectory-Lab
<h2>Description</h2>
Project consists of Downloading Oracle Virtualbox, a windows 10 ISO, and a server 2019 ISO file. Then I will create a virtual machine to house active directory
then create our domain, configure NAT and Routing, then set up a DHCP on the domain controller then before we create our client machine we need to run a 
powershell script to generate a thousand users in active directory. Lastly I will build a virtual machine with windows 10  connect it to the private virtualbox network
naming the machine "client1" we will join it to the domain and log in with one of the domain accounts.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
